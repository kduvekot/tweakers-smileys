Animation Breakdown - Worshippy Style (Unified)

Eyes and mouth animate together as one group. Scale affects both elements.

Keyframe Frame Unified Transform (Eyes + Mouth) Description
0% Frame 0 translate(0, 0) scale(1, 1) Neutral position, normal proportions
25% Frame 1 translate(0, 1px) scale(1, 1) Slight nod down, normal proportions
50% Frame 2 translate(0, 4.5px) scale(1, 0.5) Maximum nod - eyes AND mouth squish!
75% Frame 3 translate(0, 1px) scale(1, 1) Returning to slight nod (same as frame 1)
100% Frame 0 translate(0, 0) scale(1, 1) Back to neutral (loop)

Comparison with worshippy.svg:

  • worshippy.svg: Uses translate(0, 8px) scale(1, 0.5) - goes down 8px
  • yes.svg (unified): Uses translate(0, 4.5px) scale(1, 0.5) - goes down 4.5px
  • Reason: Worshippy is "worship" motion (bowing deeply), yes is gentle nodding
