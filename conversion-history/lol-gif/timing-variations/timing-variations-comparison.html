    0%, 100%       { transform: translate(0, 0px)     scale(1, 1)    skewX(0)      }  /* Up */

Pattern: Up → 1/3 down → 2/3 down → Full down → 2/3 down → 1/3 down → Up (repeat)
Effect: Smooth continuous bouncing motion matching the GIF's 6-frame cycle
