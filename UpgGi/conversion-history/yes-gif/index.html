Animation Specifications:

  • Duration: 0.7s (700ms)
  • Keyframes: 7 animation steps
  • Transform: translateY() for vertical nodding motion
  • Timing: Linear easing
  • Accessibility: Respects prefers-reduced-motion

Keyframe Breakdown:

  • 0% - Neutral position (translateY 0px)
  • 14.29% - Start nod down (translateY 0.5px)
  • 28.57% - Maximum down (translateY 1px)
  • 42.86% - Return to neutral
  • 57.14% - Passing through neutral
  • 71.43% - Nod up (translateY -0.5px)
  • 85.71%, 100% - Return to neutral

üîç Technical Details

Animation Pattern

The yes.svg uses CSS-based animation following Tweakers' standard pattern as documented in ANIMATION_COMPARISON.md.

Tools Used

  • ImageMagick - GIF frame extraction and manipulation
  • Python (Pillow) - Image analysis and comparison
  • Inkscape - SVG rendering for validation
  • Manual SVG coding - All paths hand-crafted for precision

Lessons Learned

  • Pixel-perfect matching requires multiple iterations
  • Subtle effects like squinting need careful tuning
  • Overlay comparisons are essential for validation
  • CSS animations provide better control than SMIL
  • Accessibility (prefers-reduced-motion) should be built-in from the start
